### 我相信的几件事

**做出来比说出来有用。** 我不写「AI 可能会怎样」，只放「我用 AI 做出了什么」——每一个链接都点得开。

**稀缺的是判断力，不是生成。** AI 能写无限的代码，知道该做什么、做成什么样，仍然得靠人。

**一人，全栈，三语。** 从 3D 引擎到支付到部署，从中文到日文到英文，一个人 + AI 就能端到端交付。
