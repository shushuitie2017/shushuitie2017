### 信じていること

**作ることは語ることに勝る。** 「AI に何ができるか」ではなく「AI で何を作ったか」を出す。リンクはすべて動く。

**希少なのは生成ではなく判断力。** コードは無限に生成できる。何をどう作るかは人が決める。

**一人でフルスタック、三言語。** 3D エンジンから決済・デプロイまで、中日英すべて、一人＋AI で完結。
